## 🧠 Overview

This repository documents my **9-month intensive Full Stack Web Development** program at the **Information Technology Institute (ITI)** — one of the most rigorous and comprehensive software engineering training programs in the region.

The journey spans the entire modern web development ecosystem: from core **HTML/CSS/JavaScript** fundamentals through advanced **frameworks**, **backend architectures**, **database design**, **DevOps**, and **AI-powered development** — culminating in production-grade, real-world project work.

**What you'll find here:**

- 📚 **Structured learning modules** across 30+ technologies
- 🧪 **Hands-on labs** with real-world problem-solving
- 🏗️ **Full-stack projects** demonstrating end-to-end development
- 📝 **Code samples & exercises** organized by topic and difficulty

## 🧩 What I Learned

### Frontend Development
- Building **semantic, accessible** HTML5 structures with responsive layouts
- Crafting pixel-perfect UIs with **CSS3**, Flexbox, Grid, and animations
- Writing modern **JavaScript (ES6+)**: closures, promises, async/await, modules
- Developing **TypeScript** applications with strict type safety and generics
- Building SPAs with **Angular** (modules, services, guards, interceptors, RxJS)
- Creating dynamic UIs with **React** (hooks, context, custom hooks, memoization)
- Working with **Vue.js** Composition API and reactivity primitives
- Server-side rendering and static generation with **Next.js**
- Fine-grained reactivity patterns with **Solid.js**
- Form validation and management with **React Hook Form**
- Responsive design principles and mobile-first development

### Backend Development
- Designing **RESTful APIs** with proper HTTP semantics and status codes
- Building scalable servers with **Node.js** and **Express**
- Enterprise backend architecture with **NestJS** (modules, providers, guards, pipes)
- **JWT authentication** and role-based access control (RBAC)
- API documentation and testing with **Postman**
- **GraphQL** schema design, resolvers, and query optimization

### Databases
- Relational database design and complex **SQL** queries with **PostgreSQL**
- **NoSQL** data modeling and aggregation pipelines with **MongoDB**
- ORM integration with **TypeORM** (entities, migrations, relations)

### DevOps & Testing
- **Docker** containerization: Dockerfiles, Compose, multi-stage builds
- **Unit testing** with testing frameworks and mocking strategies
- **Test-Driven Development (TDD)** workflow and best practices

### Architecture & Best Practices
- **Object-Oriented Programming**: SOLID principles, inheritance, polymorphism
- **Design Patterns**: Singleton, Factory, Observer, Strategy, and more
- **Clean Code** practices: readability, maintainability, refactoring
- State management with **NgRx** (store, effects, selectors) and **TanStack Query**

### Advanced Topics
- Building **Progressive Web Applications** with service workers and caching
- **RAG (Retrieval-Augmented Generation)** and **Agentic AI** concepts
- Modern **Vibe Coding** approaches and AI-assisted development
