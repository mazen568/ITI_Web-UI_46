## 💼 Graduate Job Profiles

| Role | Focus |
|---|---|
| 🟢 **MEAN Stack Developer** | Develops websites and web applications using the MEAN stack; JavaScript on both client and server side |
| 🔵 **MERN Stack Developer** | Full-stack applications using React.js (frontend), Node.js & Express.js (backend), and MongoDB (database) |
| 🎨 **Frontend Web Developer** | Builds the frontend of websites using HTML, CSS, and JavaScript |
| 🧩 **Full Stack Web Developer** | Builds frontend and backend using frameworks like Angular, React, Vue.js, and Node.js |
| 🖌️ **UI/UX Designer** | Creates positive user experiences and visually appealing interfaces — UX for functionality, UI for visual design |
