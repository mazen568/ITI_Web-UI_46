## ЁЯзй What I Learned

**Markup, Styling & UI**
Semantic HTML5, responsive/mobile-first layouts, CSS preprocessors (Sass/Less), utility-first styling with Tailwind, and component libraries like MUI.

**JavaScript & TypeScript**
Modern ES6+ syntax, advanced JS patterns, static typing with TypeScript, legacy DOM work with jQuery, and core CS fundamentals тАФ OOP, design patterns, and data structures.

**Front-End Frameworks**
Building SPAs and component-driven UIs with React, Angular, Vue, and SolidJS; server-side rendering with Next.js; form handling with React Hook Form; state and server-state management with NgRx and TanStack Query; Progressive Web Apps.

**Back-End Development**
Server-side development with Node.js, structured backend architecture with NestJS, and API design with GraphQL.

**Databases**
Relational modeling and SQL with PostgreSQL, and NoSQL data modeling.

**Testing**
Unit testing, test-driven development, and end-to-end testing with Playwright.

**DevOps & Tooling**
Containerization with Docker and version control fundamentals with Git.

**AI-Assisted Development**
RAG and agentic AI concepts, and modern AI-assisted ("vibe") coding workflows.
